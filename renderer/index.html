Gate detections by speaker match
If on, wake-word detections whose voice embedding doesn't match the enrolled print are silently dropped. Useful when other people can reach the mic.
Match threshold
Minimum cosine similarity required between a wake-word detection's voice embedding and the enrolled print. 0.65 is the default (balanced). Drop to 0.4 to be more forgiving; raise to 0.85+ for strict matching.
